# XCAssets Viewer

VSCode extension to view .xcassets content like in Xcode.

## Features

- View image sets with all scale variants (@1x, @2x, @3x)
- View color sets with light/dark mode variants
- View data sets
- Right-click context menu for .xcassets directories

## Usage

1. Right-click any .xcassets directory in Explorer
2. Select "Open XCAssets Viewer"
3. View assets in webview panel

## Development

```bash
npm install          # Install dependencies
npm run compile      # Compile TypeScript
npm run watch        # Watch mode
npm run lint         # ESLint
npm test             # Run tests
```

Press F5 to launch Extension Development Host.
